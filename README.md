# Computación Matemática

Repositorio de material de apoyo para la asignatura de Computación Matemática, incluyendo ejemplos prácticos, implementación computacional y ejercicios relacionados con optimización, regresión lineal, diseño de algoritmos y computación matricial.


## Contenido

### Programación lineal

Material relacionado con:

- Modelos de programación lineal
- Método simplex
- Modelos de transporte
- Solver en Excel
- Optimización con Python y MATLAB

📂 Carpeta:
[programacion-lineal](programacion-lineal)



### Regresión lineal

Material relacionado con:

- Fundamentos de regresión
- Método de mínimos cuadrados
- Regresión lineal simple
- Visualización de datos
- Modelos predictivos básicos

📂 Carpeta:
[regresion-lineal](regresion-lineal)



### Diseño de algoritmos

Material relacionado con:

- Problema del agente viajero
- Problema de la mochila
- Optimización combinatoria
- Algoritmos de búsqueda y evaluación

📂 Carpeta:
[diseno-algoritmos](diseno-algoritmos)



### Computación matricial

Material relacionado con:

- Métodos iterativos
- Método de Jacobi
- Sistemas de ecuaciones lineales
- Aproximación numérica

📂 Carpeta:
[computacion-matricial](computacion-matricial)
